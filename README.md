# Shadowrocket Config

## How to use
Open [Shadowrocket](https://www.shadowrocketdownload.com) and add configuration:
 - from url: `https://raw.githubusercontent.com/dlisin/shadowrocket-config/master/shadowrocket.conf`
 - or scan the QR code below
